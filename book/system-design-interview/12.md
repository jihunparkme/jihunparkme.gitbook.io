# 1단계: 문제 이해 및 설계 범위 확정

지원해야 하는 기능

- 1:1, 그룹 채팅 모두 지원하는 앱(응답지연이 낮은 일대일 채팅 기능)
- 모바일, 웹 모두 지원
- 일별 능동 사용자 수(DAU, Daily Active User) 기준 5천만(50million)명 처리
- 최대 100명까지 그룹 채팅 참여 가능
- 1:1 채팅, 그룹 채팅, 사용자 접속상태 표시 지원
- 메시지 길이는 100,000자 이하
- 채팅 이력은 영원히 보관
- 다양한 단말 지원, 하나의 계정으로 여러 단말에 동시 접속 지원
- 푸시 알림
